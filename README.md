# ak280-schematics

[日本語のREADMEはこちら](README.ja.md)

This repository contains the schematic for the AK-280 audio amplifier purchased from Amazon, created by tracing the printed patterns and components. The schematic is created using KiCad.

## Project Objective
This project was created for personal hobby purposes only. There is no specific intended purpose or goal.

## File Descriptions

* **hardware/**: Contains all KiCad schematic files and design data.
* **images/**: Contains the following circuit diagrams and related reference materials:
    * **ak280-schematic.pdf**: Circuit schematic (PDF file)
    * **fig01-topside-with-refno.png**: Top-side photo with circuit reference numbers (For viewing, layers merged)
    * **fig01-topside-with-refno.xcf**: Top-side photo with circuit reference numbers (For GIMP editing, layers preserved, compressed)
    * **fig02-bottomside-with-refno-org.png**: Bottom-side photo with circuit reference numbers (Original, for viewing, layers merged)
    * **fig02-bottomside-with-refno-org.xcf**: Bottom-side photo with circuit reference numbers (Original, for GIMP editing, layers preserved, compressed)
    * **fig03-bottomside-with-refno-mirror.png**: Bottom-side photo with circuit reference numbers (Mirrored, for viewing, layers merged)
    * **fig03-bottomside-with-refno-mirror.xcf**: Bottom-side photo with circuit reference numbers (Mirrored, for GIMP editing, layers preserved, compressed)
    * **fig04-ai-generated-reference.png**: Pattern diagram generated by AI and verified by visual inspection.
* **LICENSE**: Project license information.
* **README.ja.md**: Detailed project description in Japanese.
* **README.md**: Detailed project description in English (this file).

## Usage
1. Clone (download) this repository to your local environment.
2. Launch KiCad (v7.0 or later recommended) and open `hardware/AK-280.kicad_pro`.
3. Open the schematic editor to view or edit the created circuit diagram.
4. GIMP is required to view or edit the `.xcf` files in the `images/` folder.
